\begin{abstract}

La programación de sistemas de información se desarrolla utilizando lenguajes de propósito general tales como Java o C\#, y lenguajes de propósito espécifico como SQL o JPQL.
Existan una variedad de aproximaciones, tecnologías y herramientas que facilitan La construcción de estos lenguajes.
Recientemente, algunas de estas aproximaciones se han enfocado en el uso de tecnologías basadas en modelos para la Ingeniería de Lenguajes.
Este documento introduce el tema de Ingeniería de Lenguajes basados en modelos, presenta algunos de los conceptos más relevantes y muestra un ejemplo usando Eclipse Modeling Framework y EMFText.     

\end{abstract}
